What does Beta mean?

The Beta release of a software platform means it's still in the testing phase. It's functional enough to use, but is still unpolished and lacks many of the features of the production release.

What should you expect and how can you help?

While using the Beta release, you may come across usability issues and/or errors. These are called bugs and we want to know about them! All of them. We want you to let us know about the big ones along with the small ones. As a user of the Beta release, you are helping us improve the platform so that in future releases a user can have a great experience.
